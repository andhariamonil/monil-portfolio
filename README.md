# Developer Portfolio

A modern, responsive **React + Vite** portfolio to showcase  projects, skills, and certifications.  

## Tech Stack

- **Frontend**: React 18, Vite
- **Routing**: React Router DOM
- **Styling**: Custom CSS (`src/styles/styles.css`)
- **Deployment**: Vercel (`vercel.json` rewrite for SPA routing)

## Features

- **Home page** with an introduction and quick links.
- **Projects** listing with detailed project pages.
- **Skills** section with categorized skills and details.
- **Certifications** page with non-clickable certification cards.
- **Contact** page.
- SPA routing with clean URLs.

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

The dev server URL will be shown in the terminal (usually `http://localhost:5173`).
